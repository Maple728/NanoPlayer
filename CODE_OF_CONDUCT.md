# Contributor Covenant 行为准则

## 我们的承诺

为了营造开放、友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、可见或不可见的
残障、族裔、性征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人形象、
种族、宗教或性取向如何，参与本项目和社区的每一个人都不会受到骚扰。

我们承诺以有助于建设开放、友善、多元、包容和健康社区的方式行事和互动。

## 我们的准则

有助于为社区创造积极环境的行为包括但不限于：

* 对他人表达同理心和善意
* 尊重不同的意见、观点和经验
* 给予并优雅地接受建设性反馈
* 为我们的错误承担责任并向受影响的人道歉，并从中吸取经验
* 着眼于不仅对个人、也对整个社区最有利的事情

不可接受的行为包括但不限于：

* 使用带有性意味的言语或图像，以及任何形式的性关注或挑逗
* 嘲讽、侮辱或贬损性评论，以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可，发布他人的私人信息，例如物理或电子邮件地址
* 在专业环境中合理认定为不当的其他行为

## 执行责任

社区维护者有责任阐明并执行我们所认可的可接受行为标准，并对其认为不当、具有威胁性、
冒犯性或有害的任何行为采取适当且公平的纠正措施。

社区维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、
issue 及其他贡献，并在适当时说明理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间正式代表社区时的场合。代表社区的
情形包括使用项目的官方电子邮件地址、通过官方社交媒体账号发帖，或在线上或线下活动中担任
指定代表。

## 执行准则

社区维护者在判定违反本行为准则的后果时，将遵循以下「社区影响指引」：

### 1. 纠正

**社区影响**：使用了不恰当的语言，或其他被认定为不专业或不受欢迎的行为。

**后果**：由社区维护者私下提出书面警告，说明违规性质并解释为何不当。可能会要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为构成的违规。

**后果**：就持续的行为发出警告。在指定时间内，不得与相关人员互动，包括主动与执行准则者
互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区准则，包括持续的不当行为。

**后果**：在指定时间内，禁止与社区进行任何形式的互动或公开沟通。违反这些条款可能导致
永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区准则的行为模式，包括持续的不当行为、对个人的骚扰，或对某类
群体的攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
